## 4. Market vs Model + Edge Berekening

**Implied probabilities:**
- Market ADM 1.25 → implied ~80%
- Model ADM 66% → fair 1.52

**Edge Cobolli:**
- Model prob 34% vs market implied ~24.4% → **+9.6% edge**

**Fair odds Cobolli ~2.94** vs market **4.10**

**Waarom geen grote value op ADM?**
- Market is al erg kort op ADM (1.25). De edge op ADM is negatief (-14%).
- Dit is klassieke "name value" pricing. De market overschat ADM's kans omdat hij de bekendere speler is met clean pad.

**Keuze onderbouwing (stake & type):**
- Edge ~9-10% is "small value" (niet 15%+).
- Daarom **SMALL VALUE + LIVE HEDGE** aanbevolen in plaats van full stake.
- Live hedge omdat Cobolli in set 1-2 nog fris kan zijn (agressie kan dan werken). Als hij later inzakt → hedge meer.

## 6. Aanbevolen Bet & Volledige Onderbouwing van Keuzes

**Pick:** **Flavio Cobolli** (value side)  
**Odds threshold:** 3.60+ (4.10 is prima)  
**Stake:** 0.5 – 0.75 units (of 1-1.5% bankroll) – **small** omdat edge medium is en fatigue risico twee kanten op kan werken.  
**Hedge plan:**  
- Pre-match: volle small stake op Cobolli @4.10+  
- Live: als Cobolli eerste set wint of goed beweegt → hedge 30-50% (om lock in profit of break even).  
- Als ADM dominant is in set 1 → cut of kleine hedge op ADM live.  
**Min. fair odds:** ~3.20 (buffer)

**Volledige onderbouwing van deze keuzes:**
1. **Model vs Market discrepantie** (9.6% edge) → value bestaat.
2. **Fatigue + path data** (14 sets + five-setter + heat) → onderbouwt waarom model waarschijnlijk te optimistisch is over ADM.
3. **UE/WPU stats** → Cobolli kan in theorie aggressief zijn, maar recente toernooilast + hitte maakt dat riskant. Dit ondersteunt "small" size i.p.v. vol.
4. **Live aspect** → typisch BSP: underdogs met fatigue kunnen vroeg goed zijn, later instorten. Hedge beschermt.
5. **Bankroll management** → edge is niet mega, dus geen Kelly vol (we gebruiken lichte fraction + hedge).
6. **Risks expliciet** → we noemen de collapse risk en de mogelijkheid dat Cobolli gewoon niet goed genoeg is.

**Risico's (eerlijk benoemd):**
- Cobolli is gewoon de mindere speler en verliest clean.
- ADM's fitheid en return game maken hem te sterk in 3-4 sets.
- Weer kan Cobolli ook raken (hitte voor beide).
